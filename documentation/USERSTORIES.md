## User Stories

### Authentication
- **As a new user**, I want to create an account so that I can submit and manage road issues.
- **As a registered user**, I want to log in to my account so that I can use the reporting features of the website.
- **As a logged-in user**, I want to log out so that my data stays protected on a shared device.

### Reporting Issues
- **As a user**, I want to submit a report so that I can inform the system about potholes, damaged roads, or unsafe conditions.
- **As a user**, I want to include a description and location so that others can understand the reported issue clearly.
- **As a user**, I want to report anonymously so that I can protect my identity.

### Viewing Reports
- **As a user**, I want to see reports submitted by others so that I can identify areas with road problems.
- **As a logged-in user**, I want to see my reported issues so that I can track them.
- **As a user**, I want to see the status of reported issues, such as *reported*, *under review*, or *fixed*.

### Identifying Frequent Issues
- **As a road authority or system user**, I want to see locations reported multiple times by different users so that urgent road issues can be identified quickly.

### Admin Features
- **As an admin user**, I want to view all posted reports so that I can monitor reported issues.
- **As an admin user**, I want to modify reports so that incorrect or incomplete information can be corrected.
- **As an admin user**, I want to delete fake or untrustworthy reports so that the system remains reliable.
- **As an admin user**, I want to block or remove users who repeatedly submit false reports so that misuse of the platform can be prevented.
